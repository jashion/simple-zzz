# simple-zzz
simple-zzz是一个宠物图片分享社区

## 目录划分
* conf: 配置文件
* doc: 文档
* logs: 日志文件
* src: 源码文件  
```
    src/account: 帐号模块  
    src/client: 客户端代码
```
